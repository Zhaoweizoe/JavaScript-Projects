Responsive Text

Resize the browser window to see how the text size scales.

Use the "vw" unit when sizing the text.10vw will set the size to 10% of the viewport width.

Viewport is the browser window sizze.1vw=1% of the viewport width.If the viewport is 100cm wide,1vw is 1cm. The text size can be set with a vw unit, which means the "viewport width".
